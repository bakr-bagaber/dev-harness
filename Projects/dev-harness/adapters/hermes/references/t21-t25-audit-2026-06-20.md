## Bugs Found & Fixed

### Bug 1: Node 18 compat in test runner
- **File:** `test/run-all.mjs`
- **Problem:** Used `import.meta.dirname` (Node 21+), crashes on Node 18 (project minimum)
- **Fix:** Replaced with `fileURLToPath(import.meta.url)` pattern
- **Pattern:** Always use `resolve(dirname(fileURLToPath(import.meta.url)), '..')` for ESM path resolution when Node 18 compat is required

### Bug 2: Coverage gate missing from VERIFY phase
- **File:** `cli/lib/gates.mjs`
- **Problem:** `checkCoverage` registered only in BUILD, not VERIFY (T22 spec requires both)
- **Fix:** Added `checkCoverage` to VERIFY checks array

### Bug 3: Stub assertion in T7 test
- **File:** `test-t7.mjs`
- **Problem:** BUILD check count bumped 4→5 but `names.includes('coverage')` assertion was never added
- **Fix:** Added explicit coverage name check. Also updated VERIFY test (2→3 checks)
- **Pattern:** When adding a new gate check, always update both the count assertion AND the names-includes assertion in all affected phase tests

### Bug 4: Dead test copies
- **File:** `test/` directory
- **Problem:** 14 orphan test files (duplicates and broken copies), not used by runner
- **Fix:** Removed all; `test/` now contains only `run-all.mjs`
- **Pattern:** Runner uses root `test-t*.mjs` files — never duplicate tests into a subdirectory unless the runner is wired to use them
