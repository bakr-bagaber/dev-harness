## Issues Found

### Dead export shouldAutoPrompt()

Defined in modes.mjs:28-32 but has zero callers across the codebase. phase.mjs imports only promptYesNo and shouldConfirmGates. Config key copilot.autoPrompt is written but never read.

**Impact:** Setting autoPrompt: false has no effect. Setting confirmGates: false suppresses the prompt entirely.

### 3 remaining || instead of ??

All three in cli/ use config.mode || copilot instead of config.mode ?? copilot:

- cli/lib/ralph-outer.mjs:42
- cli/commands/phase.mjs:49
- cli/commands/phase.mjs:78

T10 fixed ralph-inner.mjs but missed these. Low practical impact (mode is always set) but inconsistent.

### No test-t11.mjs

Zero tests for getMode, shouldAutoPrompt, shouldConfirmGates, promptYesNo, ensureCopilotConfig, or set-mode command.

### T10 test I.8 outdated

test-t10.mjs expects resume to return not_implemented. T11 implemented resume (sets paused=false). Test assertion is stale.
